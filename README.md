# ADAS_SESAC

이 프로젝트는 **ADAS(Advanced Driver Assistance System) SESAC** 기능을 구현하는 Python 기반 저장소입니다.

## 프로젝트 주요 기능

- **객체 탐지:** 차선, 차량, 보행자 등 다양한 객체를 탐지하는 ADAS 핵심 알고리즘 제공  
- **시각화:** 탐지 결과의 시각화 및 분석 툴 내장  
- **데이터 전처리:** 원천 데이터셋의 COCO 포맷 변환, 거리 정보 추가 등 커스텀 데이터 처리 지원  
- **모델 학습 및 관리:** 탐지 모델의 학습, 적용 및 설정 관리 기능  
- **유연한 설정:** config.py를 활용한 다양한 설정값 조절 가능
